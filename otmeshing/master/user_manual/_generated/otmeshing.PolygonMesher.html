Examples

Triangulate a parallelogram:

>>> import otmeshing
>>> mesher = otmeshing.PolygonMesher()
>>> polyline = [[0, 0], [3, 0], [4, 2], [1, 2]]
>>> triangulation = mesher.build(polyline)
