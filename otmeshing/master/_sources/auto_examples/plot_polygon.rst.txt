Define a non-convex regular polygon from a set of points

.. GENERATED FROM PYTHON SOURCE LINES 15-24

.. code-block:: Python

    polyline = []
    n = 20
    for i in range(n):
        r = 2.0 + i % 2
        theta = i * 2.0 * math.pi / n
        x = r * math.cos(theta)
        y = r * math.sin(theta)
        polyline.append([x, y])
