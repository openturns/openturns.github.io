(Source code, png, hires.png, pdf)

class otrobopt.VarianceMeasure(*args)¶

Variance measure function.

It is built from a parametric function and a distribution of density function .

In case of a discrete distribution:

Parameters:
functionopenturns.Function

Parametric function

distributionopenturns.Distribution

Distribution of parameters

Examples

>>> import openturns as ot
>>> import otrobopt
>>> thetaDist = ot.Normal(2.0, 0.1)
>>> f_base = ot.SymbolicFunction(['x', 'theta'], ['x*theta'])
>>> f = ot.ParametricFunction(f_base, [1], [1.0])
>>> measure = otrobopt.VarianceMeasure(f, thetaDist)
>>> x = [1.0]
>>> print(measure(x))
[0.01]
