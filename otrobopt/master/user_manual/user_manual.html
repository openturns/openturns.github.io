The goal is to formulate and solve robust optimization problem.

A robust optimization problem consists of a parametric objective objective and/or a parametric inequality constraint where is a design variable and a parameter.

The problem is made robust by:

  • modelling the parameter by the the random vector with given distribution .

  • choosing measure functions and for the objective and constraint functions.

The definition of the measure functions is associated to the concept of MeasureEvaluation.

A measure evaluation can be used through MeasureFunction to expose generic function services.

A robust optimization problem can be defined with RobustOptimizationProblem, and then solved using a RobustOptimizationAlgorithm.

Note that this measure evaluation can be discretized over so as to define a deterministic optimization problem using MeasureFactory.
