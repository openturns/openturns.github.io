DomainComplement¶

class DomainComplement(*args)¶

Complement of a Domain.

A point is inside this domain if it is not contained in domain passed as argument.

Parameters:
domainDomain

Examples

>>> import openturns as ot
>>> interval = (ot.Interval([1], [2], [False], [True]))
>>> complement = ot.DomainComplement(interval)
>>> [5] in complement
True
>>> [1.5] in complement
False
