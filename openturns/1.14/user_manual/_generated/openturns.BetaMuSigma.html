Examples

Create the parameters of the Beta distribution:

>>> import openturns as ot
>>> parameters = ot.BetaMuSigma(0.2, 0.6, -1, 2)

Convert parameters into the native parameters :

>>> print(parameters.evaluate())
[2,3,-1,2]
