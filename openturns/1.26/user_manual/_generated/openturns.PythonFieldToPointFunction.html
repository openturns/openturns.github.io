Examples

For example, we create the function that maps a field to the mean value of its values. The values of the input field are of dimension and the output value of the function is also of dimension .

>>> import openturns as ot
>>> mesh = ot.Mesh(1)
>>> def myPyFunc(X):
...     Y = ot.Sample(X).computeMean()
...     return Y
>>> inputDim = 2
>>> outputDim = 2
>>> myFunc = ot.PythonFieldToPointFunction(mesh, inputDim, outputDim, myPyFunc)
