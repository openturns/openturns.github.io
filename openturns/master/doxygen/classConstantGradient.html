Detailed Description

Class for a linear numerical math gradient impelmentation.

This class offers an interface for an real numerical mathematical gradient implementations into the platform. It realizes the computation of the image matrix of a vector (aka Point) through the gradient. The gradient may be loaded in an external way through a wrapper.
