Detailed Description

Abstract top-level class for all numerical math function implementations.

The class that simulates a numerical math function, its gradient and its hessian. This class is just an interface to actual implementation objects that can be hot-replaced during computation. Each implementation object refers to the evaluation, the gradient or the hessian.
