Detailed Description

This class defines load and save mechanisms.

This is the base class of all objects that can be stored into and reloaded from the study.

The class defines an unique Id for every object so they can be equal but not identical. This Id is an essential part for the management of objects in studies.

CLASSNAME PersistentObject ( )

Default constructor

The constructor sets a new Id to the object, so it can be later referenced by a Study object. It is also declared visible if member of a study.

The object has the default name but it does not use storage for it.
