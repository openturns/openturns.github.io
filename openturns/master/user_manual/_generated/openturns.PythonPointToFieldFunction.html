Examples

For example, we create the function which maps the point into the field where the output values are on the regular grid .

>>> import openturns as ot
>>> mesh = ot.RegularGrid(0.0, 0.1, 11)
>>> def  myPyFunc(X):
...     size = 11
...     Y = [ot.Point(X)*i for i in range(size)]
...     return Y
>>> inputDim = 2
>>> outputDim = 2
>>> myFunc = ot.PythonPointToFieldFunction(inputDim, mesh, outputDim, myPyFunc)

>>> Yfield = myFunc([1.1, 2.2])
