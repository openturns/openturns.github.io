Examples

Create the parameters of the LogNormal distribution:

>>> import openturns as ot
>>> parameters = ot.LogNormalMuErrorFactor(0.63, 1.5, -0.5)

Convert parameters into the native parameters:

>>> print(parameters.evaluate())
[0.63,0.246505,-0.5]
