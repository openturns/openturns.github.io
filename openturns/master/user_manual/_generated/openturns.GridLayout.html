add(grid)¶

Add subgraphs from a different grid to the corresponding subgraphs in the current grid.

Parameters:
gridGridLayout

GridLayout containing the subgraphs to add.

Notes

Only works if both grids have the same number of rows and columns, because the (i,j)-th graph of the provided grid is added to the (i,j)-th graph of the original grid with the add() method.

setLegendCorner(corner)¶

Accessor to the corner of the legend of each subgraph.

Parameters:
cornersequence of float

Relative coordinates (in [0,1]x[0,1]) of the legend in the graph bounding box.

Notes

This enables the legend to be located outside the graph.

Examples

>>> import openturns as ot
>>> grid = ot.GridLayout(1, 3)
>>> for j in range(grid.getNbColumns()):
...    beta = 1.0 + j
...    grid.setGraph(0, j, ot.Gumbel(beta, 0.0).drawCDF())
>>> grid.setLegendCorner([0.5, 0.5]) # legend will begin in the middle
