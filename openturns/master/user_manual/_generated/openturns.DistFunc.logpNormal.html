logpNormal¶

logpNormal(x, tail=False)¶

Log-CDF of a unit-variance centered Normal distribution.

Parameters:
xfloat

Location.

tailbool, optional

Tail flag. Default value is False. If True, the log-complementary CDF is computed.

Returns:
logCdffloat

The log-CDF or the log-complementary CDF at .

Examples

>>> import openturns as ot
>>> logCdf = ot.DistFunc.logpNormal(0.9)
