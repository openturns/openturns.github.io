ProfileLikelihoodResult¶

class ProfileLikelihoodResult(*args)¶

Distribution factory result for profile likelihood estimation.

This result class provides the additional optimal likelihood value and the confidence interval of the parameter according to the deviance statistic (refer to HypothesisTest::LikelihoodRatioTest()).

Parameters:
distributionDistribution

Estimated distribution

parameterDistributionDistribution

Distribution of the parameters

logLikelihoodfloat

Optimal log-likelihood value

profileLikelihoodFunction

Profile likelihood function

parameterfloat

Optimal outermost parameter value

See also

GeneralizedExtremeValueFactory
