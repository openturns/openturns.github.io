Examples

Create the parameters of the Gumbel distribution:

>>> import openturns as ot
>>> parameters = ot.GumbelLambdaGamma(2.0, 0.5)

Convert parameters into the native parameters :

>>> print(parameters.evaluate())
[0.5,0.5]
