Notes

Let be a functional basis of size , where each function with . Let be some input vectors in .

The class builds the matrix defined by:

The matrix columns are effectively built when the computeDesign method is activated. The values are stored in a cache. Can be useful for an iterative least squares problem resolution or in interaction with LeastSquaresMethod to select the algorithm used for the resolution of linear least-squares problems.

It is posisble to directly specify the values in the last constructor.

Examples

>>> import openturns as ot
>>> basisSize = 3
>>> sampleSize = 5
>>> X = ot.Sample(sampleSize, 1)
>>> X = ot.Sample.BuildFromPoint(range(1, 1 + sampleSize))
>>> func_coll = []
>>> for j in range(basisSize):
...     func_coll.append(ot.SymbolicFunction(['x'], ['x^' + str(j + 1)]))
>>> basis = ot.Basis(func_coll)
>>> proxy = ot.DesignProxy(X, func_coll)
>>> proxy2 = ot.DesignProxy(X, basis)
