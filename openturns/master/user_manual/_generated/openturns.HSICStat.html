HSICStat¶

class HSICStat(*args)¶

Base class of HSICStat.

See also

HSICUStat, HSICVStat

Notes

This is a base class for the implementation of an HSIC statistic but cannot be used by itself. An HSIC statistic object can be created by using the derived classes: HSICUStat and HSICVStat.
