class KFold(*args)¶

K-fold.

Parameters:
kpositive integer

Number of folds in which the sample is split. If not provided, default is .

See also

FittingAlgorithm, CorrectedLeaveOneOut

Notes

KFold inherits from FittingAlgorithm.

This class is not usable on its own because it has sense only within the FunctionalChaosAlgorithm.
