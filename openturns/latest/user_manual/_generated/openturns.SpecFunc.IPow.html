IPow¶

IPow(*args)¶

Raise the given to the integral power .

Parameters:
mint
nint
xfloat
Returns:
resultint or float

The result has the same type as the first argument.

Examples

>>> import openturns as ot
>>> ot.SpecFunc.IPow(-2.5, 3)
-15.625
>>> ot.SpecFunc.IPow(5, 3)
125
