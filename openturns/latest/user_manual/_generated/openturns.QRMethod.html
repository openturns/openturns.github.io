Examples

Solves a linear least squares problem with SVD method:

>>> import openturns as ot
>>> A = ot.Matrix([[1, 1], [1, 2], [1, 3], [1, 4]])
>>> y = [6, 5, 7, 10]
>>> method = ot.QRMethod(A)
>>> x = method.solve(y)
>>> print(x)
[3.5,1.4]
