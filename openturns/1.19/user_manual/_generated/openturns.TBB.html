TBB¶

class TBB(*args, **kwargs)¶

Threading parameters.

The number of threads is the value associated to the TBB-ThreadsNumber key in ResourceMap, which is initialized to the available cpus, and can be overriden by the OPENTURNS_NUM_THREADS environment variable.
