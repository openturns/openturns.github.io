Examples

Create the parameters of the WeibullMin distribution:

>>> import openturns as ot
>>> parameters = ot.WeibullMinMuSigma(1.3, 1.23, -0.5)

Convert parameters into the native parameters :

>>> print(parameters.evaluate())
[1.99222,1.48961,-0.5]
