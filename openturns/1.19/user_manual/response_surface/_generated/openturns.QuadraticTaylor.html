QuadraticTaylor¶

class QuadraticTaylor(*args)¶

Second order polynomial response surface by Taylor expansion.

Available constructors:

QuadraticTaylor(center, function)

Parameters
centersequence of float

Point where the Taylor expansion of the function is performed.

functionFunction

Function to be approximated.

See also

LinearTaylor, LinearLeastSquares, QuadraticLeastSquares

Notes

The approximation of the model response around a specific set of input parameters may be of interest. One may then substitute for its Taylor expansion at point . Hence is replaced with a first or second-order polynomial whose evaluation is inexpensive, allowing the analyst to apply the uncertainty anaysis methods.

Examples

>>> import openturns as ot
>>> formulas = ['x1 * sin(x2)', 'cos(x1 + x2)', '(x2 + 1) * exp(x1 - 2 * x2)']
>>> myFunc = ot.SymbolicFunction(['x1', 'x2'], formulas)
>>> myTaylor = ot.QuadraticTaylor([1, 2], myFunc)
>>> myTaylor.run()
>>> responseSurface = myTaylor.getMetaModel()
>>> print(responseSurface([1.2,1.9]))
[1.13655,-0.999155,0.214084]
