class QuadraticBasisFactory(*args)¶

Quadratic basis factory.

Examples

>>> import openturns as ot
>>> basis = ot.QuadraticBasisFactory(2).build()
>>> f = ot.AggregatedFunction(basis)
>>> x = [2, 3]
>>> print(f(x))
[1,2,3,4,6,9]
