pNormal¶

pNormal(*args)¶

Cumulative distribution function of a unit-variance centered Normal distribution.

Parameters:
xfloat

Location.

tailbool, optional

Tail flag. Default value is False. If True, the complementary CDF is computed.

Returns:
pfloat

The CDF or the complementary CDF at point .

Examples

>>> import openturns as ot
>>> cdf = ot.DistFunc.pNormal(0.9)
