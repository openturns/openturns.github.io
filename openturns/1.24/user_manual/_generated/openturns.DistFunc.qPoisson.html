qPoisson¶

qPoisson(_lambda, p, tail=False)¶

Quantile of a Poisson distribution.

Parameters:
lambdafloat,

The intensity of the Poisson distribution.

pfloat, in

The probability.

tailbool, optional

Tail flag. Default value is False. If True, the quantile associated to is computed.

Returns:
qfloat

The quantile of order or .

Examples

>>> import openturns as ot
>>> q = ot.DistFunc.qPoisson(2.0, 0.95)
