class VertexValueFunction(*args)¶

Notes

Vertex value functions act on fields to produce fields such that:

with , and a mesh of .

A vertex value function keeps the mesh unchanged: the input mesh is equal to the output mesh.

The field is defined by the function :

When is not specified, the constructor builds an object which evaluation operator is not defined (it throws a NotYetImplementedException). The instanciation of such an object is used to extract an actual VertexValueFunction from a Study.

Examples

>>> import openturns as ot

Create a function such as:

>>> h = ot.SymbolicFunction(['t', 'x'], ['x + t^2'])

Convert into a vertex value function with :

>>> n = 1
>>> tg = ot.RegularGrid(0.0, 0.2, 6)
>>> myVertexValueFunction = ot.VertexValueFunction(h, tg)
>>> # Create a TimeSeries
>>> data = ot.Sample(tg.getN(), h.getInputDimension()-1)
>>> for i in range(data.getSize()):
...     for j in range(data.getDimension()):
...         data[i, j] = i * data.getDimension() + j
>>> ts = ot.TimeSeries(tg, data)
>>> print(ts)
    [ t   v0  ]
0 : [ 0   0   ]
1 : [ 0.2 1   ]
2 : [ 0.4 2   ]
3 : [ 0.6 3   ]
4 : [ 0.8 4   ]
5 : [ 1   5   ]
>>> print(myVertexValueFunction(ts))
    [ y0   ]
0 : [ 0    ]
1 : [ 1.04 ]
2 : [ 2.16 ]
3 : [ 3.36 ]
4 : [ 4.64 ]
5 : [ 6    ]
